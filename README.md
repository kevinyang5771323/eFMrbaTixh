```markdown
# 项目简介

## 核心功能
- 首页歌单推荐
- 歌单分类分页
- 歌手分类分页
- 查看收藏歌曲
- 搜索音乐（歌手、歌曲、歌单名）
- 用户信息修改
- 歌曲播放、进度条拉伸、歌词加载
- 歌曲收藏
- 歌曲下载
- 系统首页展示统计数据
- 用户管理
- 歌手管理
- 歌曲管理（修改音源、歌词、后台评论）
- 上传音乐

## 适用场景
- 音乐平台
- 音乐推荐系统
- 音乐管理系统

## 技术特点
- 后端框架：Springboot，Mybatis
- 前端技术：Vue，ElementUI，ECharts
- 开发环境：JAVA版本：jdk1.8，IDEA运行
- 数据库类型：mysql8
```
